## Getting Started
To run the project locally, follow these steps:
### Prerequisites
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation
1. Clone the repo
   ```js
   git clone https://github.com/AyushIyankan/Factly-Web-App
   ```
3. Install NPM packages
   ```js
   npm install
   ```
4. Start project
   ```js
   npm start
   ```
